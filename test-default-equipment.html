Test Default Equipment and Starting Gold

Manual Testing Instructions

  1. Go to the Character Creator
  2. Select different classes (Ranger, Rogue, Wizard, Cleric, etc.)
  3. Navigate to the Inventory & Equipment page (Page 4)
  4. Check if the Standard Package shows:
    • Equipment choices (Option A vs Option B)
    • Default equipment specific to each class (see list below)
    • Starting gold (instead of 0)
  5. Verify that making equipment choices updates the All Inventory section
  6. Confirm default equipment appears alongside chosen equipment
  7. Test multiple classes to see different default equipment

Implementation Summary

The following components were implemented:

  • Equipment Choices: Interactive Option A/B selection for class equipment
  • Default Class Equipment: Automatic inclusion of class-specific gear (Longbow + arrows for Rangers)
  • Starting Gold Display: Shows actual starting money from class/background instead of 0
  • All Inventory Integration: Combines chosen equipment, default equipment, and background items
