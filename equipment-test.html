Equipment Integration Test

Instructions

Open your browser to http://localhost:3001 and navigate to the character creator.

Select a class (e.g., Fighter) and background (e.g., Soldier), then go to step 4 (Inventory & Equipment).

Check if the equipment from your class and background appears in the inventory.

Expected Results

  • Class Equipment: Fighter should show weapons like "Longsword" or "Martial Weapons"
  • Background Equipment: Soldier should show items like "Uniform", "Belt Pouch", etc.
  • Equipment Source: Items should be marked with their source (class/background)

Debug Information

If equipment doesn't appear, check the browser console for errors and verify:

  • Class details are being loaded (check character.classDetails)
  • Background details are being loaded (check character.backgroundDetails)
  • Store methods updateClassTraits() and updateBackgroundTraits() are called
  • Character.equipment array is populated with items

Manual Console Test

You can also test in the browser console:

// Check if character store is available
const characterStore = window.characterStore || /* get from your app */;
console.log('Character:', characterStore.character);
console.log('Class Details:', characterStore.character.classDetails);
console.log('Background Details:', characterStore.character.backgroundDetails);
console.log('Equipment:', characterStore.character.equipment);
